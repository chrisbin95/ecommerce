ZeeCart is a high-performance, stylish e-commerce platform designed with a 2026 "Glassmorphism" UI. It features advanced routing, dynamic category filtering, and an immersive user experience built on React.

---

## üöÄ Key Features

* **Premium UI/UX**: Midnight & Orange aesthetic with high-end animations and glassmorphism.
* **Modern Navigation**: Sticky navbar with smart active-state tracking and animated underlines.
* **Dynamic Collections**: Automated filtering system for specialized categories via URL parameters.
* **Ninja Hub**: A custom User Dashboard (`/user`) for managing orders and account status.
* **Immersive Transitions**: Full-screen brand loader for a seamless "powering up" experience.
* **AI Integrated**: Support for OpenAI-powered customer service chat.

## üì¶ Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/chrisbin95/ecommerce.git](https://github.com/chrisbin95/ecommerce.git)
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Configure Environment Variables**
    Create a `.env` file in the root directory and add your keys:
    ```env
    REACT_APP_OPENAI_KEY=your_key_here
    ```

4.  **Start Development Server**
    ```bash
    npm start
    ```
